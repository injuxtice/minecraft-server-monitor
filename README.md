# minecraft-server-monitor
monitors when a player joins or leaves

# usage
`python3 monitor.py -ip {ip address} -port {port number}`

## example
`python3 monitor.py -ip 127.0.0.1 -port 25565`

# requirements
- python3
- mcstatus
```python3 -m pip install mcstatus```
